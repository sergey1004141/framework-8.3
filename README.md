# framework-8.3 (Мой спмописный фреймворк)

## Используемые быблиотеки:

>- Buggregator - отладочная библиотека в проекте работают тукае функции как dump() и dd()
>- Vlucas/Valitron - Библиотека валидации интерированна в базовую молдель проекта

## Роутер:
> Роутер одноуровневый.\
> Все контроллеры должны находиться в папке "app/controllers"\
> Для добавления маршрутов на метод вешается атрибут
> Пример: #[Route(path: '/register', method: 'GET')]
> Возможные параметры:
> - path: принимает строку '[указывается путь адресной строки]'
> - method: принимает строку или массив строк ('GET' или ['GET', 'POST'])
